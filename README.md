# 사이먼에듀 말씀 암송 플랫폼 (Simon Edu Scripture Memorization Platform)

성경 구절을 빈칸 채우기 퀴즈 형식으로 암송하고 연습하며, 매일의 출석체크와 실시간 랭킹 시스템을 통해 지속적으로 말씀을 마스터해 나가는 인터랙티브 웹 플랫폼입니다.

---

## 🌟 주요 기능 (Key Features)

### 1. 오늘의 말씀 암송 미션 (Daily Mission)
* **단계별 말씀 암송**: 매일 지정된 구절을 바탕으로 빈칸을 채워나가며 암송 시험을 치릅니다.
* **실시간 달성도 피드백**: SVG 기반 원형 프로그레스 바를 통해 현재 성공률을 실시간으로 확인합니다.
* **성공 및 실패 인터랙션**: 성공 시 캔버스 콘페티(폭죽 효과)와 포인트 플로팅 효과가 제공되며, 실패 시 하트 개수(기회 3회)와 타이머가 연동됩니다.

### 2. 말씀 암송 테스트 모드 (Test Mode) [v1.2.0]
* **자율 선택 연습**: 유저의 진행도나 오늘 미션 완료 여부와 관계없이, 요한계시록 7장 1절부터 17절까지 원하는 구절을 자유롭게 선택하여 테스트를 치를 수 있습니다.
* **포인트 미지급 연습 모드**: 테스트 모드에서는 헤더에 `[테스트]` 표기가 붙고 포인트 UI가 가려지며, 완료 후에도 Firestore 포인트나 개인 진행률에 영향을 주지 않아 순수한 연습용으로 적합합니다.

### 3. 출석 체크 챌린지 및 이력 조회 (Attendance Challenge)
* **일자별 출석 도장**: 매일 1회 출석 체크를 할 수 있는 인터랙티브 달력 컴포넌트가 탑재되어 있습니다.
* **연속 출석 보너스 스트릭**: 5일(150P), 10일(200P), 15일(250P), 30일(300P) 연속 출석 시 추가 보너스 포인트를 획득합니다.
* **상세 누적 출석 이력 조회 [v1.1.0]**: `총 누적 출석: N일` 배지를 클릭하면, 역대 출석한 날짜들이 요일을 포함한 한국어 형식(예: `5회차: 2026년 5월 22일 (금)`)으로 파싱되어 최근 순으로 부드러운 애니메이션과 함께 리스트업됩니다.
* **자가 치유 기능 (Self-Healing)**: 타임존 차이 등으로 유저의 캘린더에 과거 출석 일자가 누락되었지만 연속 출석(Streak) 데이터가 남아있는 경우, 유저 정보 로드 시 누락된 일자들을 자동으로 복원하여Firestore에 동기화합니다.

### 4. 명예의 전당 (Top 10) & 전체 회원 랭킹 [v1.2.0]
* **실시간 리더보드 (Top 10)**: 획득 포인트를 기준으로 정렬된 공동 순위(Joint Rank, 1-2-2-4 방식) 시스템이 탑재되어 있습니다.
* **실시간 나의 랭킹 카드**: 본인의 현재 전체 등수와 상위 백분율(%) 정보를 대시보드에서 실시간으로 확인합니다.
* **전체 회원 랭킹 팝업 모달**:
  - 나의 랭킹 카드를 클릭하거나 명예의 전당 하단의 "전체 랭킹 보기" 버튼을 누르면 글래스모피즘 스타일의 전체 랭킹 팝업이 나타납니다.
  - **Premium Detail (자동 스크롤)**: 본인의 등수가 10등 이하에 있어 스크롤 영역 밖에 있더라도, 팝업이 열릴 때 내 등수 행(`.all-ranking-item.me`) 위치로 스크롤바가 **자동으로 부드럽게 스크롤**됩니다.
  - **실시간 검색 기능**: 팝업 내 검색창을 통해 전체 유저의 순위와 포인트를 이름으로 즉시 필터링 및 검색할 수 있습니다.

### 5. 실시간 동기화 및 포인트 시스템
* **Firebase Firestore Real-time Sync**: 다른 사용자가 포인트를 얻거나 출석을 할 때마다 대시보드 리더보드와 전체 랭킹 팝업이 새로고침 없이 실시간으로 정렬 및 업데이트됩니다.
* **포인트 플로팅 효과**: 미션을 성공할 때마다 대시보드 화면 중앙에 감각적인 `+N P` 플로팅 텍스트가 무작위 좌표로 피어오릅니다.

## 🛠 기술 스택 (Tech Stack)

* **Frontend**: HTML5, Vanilla CSS, Vanilla JavaScript (ES6)
* **Database & Auth**: Firebase Auth, Cloud Firestore (Compat SDK v10.8.0)
* **Hosting**: Firebase Hosting (classic static CDN)
* **Libraries**: Canvas Confetti (Confetti폭죽 효과), Google Material Icons Round (아이콘 시스템)

## 🚀 로컬 실행 방법 (How to Run Locally)

1. 이 저장소를 클론하거나 다운로드합니다.
   ```bash
   git clone https://github.com/aiKyungHoon/Simon_Edu.git
   cd Simon_Edu
   ```
2. 로컬 웹 서버를 구동하여 실행합니다. (파이어베이스 호스팅 리디렉션 및 리소스를 정상적으로 로드하기 위해 로컬 서버 구동을 권장합니다.)
   * **Node.js (http-server 등)**:
     ```bash
     npx http-server .
     ```
   * **Python**:
     ```bash
     python -m http.server 8000
     ```
3. 브라우저에서 제공되는 로컬 주소(예: `http://localhost:8080`)로 접속합니다.

## 📦 배포 방법 (Deployment)

본 프로젝트는 Firebase Hosting에 호스팅되어 있으며, 아래 명령어를 통해 즉시 배포할 수 있습니다. (Firebase CLI가 설치되어 있고 로그인이 완료된 상태여야 합니다.)

```bash
# Firebase CLI 설치 (필요한 경우)
npm install -g firebase-tools

# Firebase 로그인
firebase login

# 호스팅 배포 진행
firebase deploy --only hosting
```

* **배포 사이트 URL**: [https://simon-edu-bible-game.web.app](https://simon-edu-bible-game.web.app)
* **캐시 무효화**: 브라우저(특히 모바일 Safari)의 강력한 캐싱으로 인해 업데이트 사항이 반영되지 않는 문제를 방지하고자, CSS 및 JS 파일 수정 배포 시 `index.html` 내의 버전 스트링(`APP_VER` 및 `?v=X.X.X` 쿼리 파라미터)을 인상하여 배포하고 있습니다.
